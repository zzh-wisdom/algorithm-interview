# 常见数据算法

## C(n,m-2, n-1)

```cpp
class Solution {
public:
    int uniquePaths(int m, int n) {
        long long int cnt = 1;
        for(int i = m, j = 1; i <= m+n-2; ++i, ++j) {
            cnt *= i;
            cnt /= j;
        }
        return cnt;
    }
};
```

cnt 需要比答案使用更长的整形

需要先乘后除，防止出现不整除的情况
